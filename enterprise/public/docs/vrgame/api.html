VR游戏

充值接口说明

1.此接口目的是为游戏提供例如购买货币、商品等支付功能

2.调用此接口，需启动VRonline平台并登录账号

3.初始接口

名称 描述
gamename 游戏名
hwnd 游戏主窗口句柄
appid 第三方 后台提交获取到的，针对没个产品唯一id
paykey 后台获取
appkey 后台获取
serverid 没有可以不传
logpath log文件存放位置 默认放在当前目录 
VRONLINAPI bool VRonline_ApiInitialize(const wchar_t* gamename,
                                        unsigned long hwnd, 
                                        const char* appid,
                                        const char* appkey,
                                        const char* paykey,
                                        const char* serverid = 0,
                                        const wchar_t* logpath = 0);

代码示例

assert(VRonline_ApiInitialize(L"Test", 
                            (unsigned long)hWnd, 
                            "1000014", 
                            "EsnG7bJuh-0I_58CaLmSaZpsjqOSzk7o", 
                            "aOWucoAiTalPC1QKgqvmRvph9wCAP_Aa"));

4.设置支付回调

名称 描述
code 返回的错误码 0是成功
msg 返回消息提示 
typedef void (VRONLINECALL *VRonline_OpenApiPayCallBackFunc)(int code, const char* msg);
VRONLINAPI void VRonline_SetPayCallBack(VRonline_OpenApiPayCallBackFunc callbackfunc);

代码示例

    void PayCallBack(int code, const char* msg)
    {
        std::cout << "ret code[" << code <<　"], msg[" << msg << "]." << std::endl;
    }
    VRonline_SetPayCallBack(PayCallBack);

5.购买道具

是否成功失败在这个回调中 VRonline_OpenApiPayCallBackFunc

名称 描述
item_id 物品id
item_price 单价
item_name 物品名
item_img_url 物品图片url
total_rmb 要充值多少rmb
is_dev 是否测试，这个要去后台管理员获取支持 
VRONLINAPI bool VRonline_PayItem(const char* item_id, 
                                    float item_price, 
                                    const char* item_name, 
                                    const char* item_img_url, 
                                    float total_rmb, 
                                    bool is_dev = false);

代码示例 ###

VRonline_PayItem("7", 
    0.01f, 
    "ceshi", 
    "http://img3.duitang.com/uploads/item/201411/07/20141107220551_TMMsE.jpeg", 
    1.0f, 
    false);
